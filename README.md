# Azure-Ideas

A curated collection of guidance, conventions, templates and examples for designing, naming and operating resources in Microsoft Azure.

## Overview

This repository gathers practical ideas and opinionated best practices for working with Azure: naming principles, resource design patterns, IaC templates, scripts and reference material to help teams make consistent decisions at scale.

## Contents

- Naming Principles: [Azure Naming Principles](./Naming/Azure_Naming_Principles_EN.md) — framework for designing resource names and conventions.
- Templates: Infrastructure-as-Code examples and deployment templates.
- Patterns & Examples: Architecture and operational patterns for common Azure workloads.
- Scripts & Tools: Helpful scripts and small utilities used when managing Azure resources.

## How to use

1. Browse the folders for the topic you need (Naming, Templates, Patterns, Scripts).
2. Read the guidance documents and adapt the examples to your naming conventions and environment.
3. Reuse templates and scripts as a starting point for automation and deployments.
